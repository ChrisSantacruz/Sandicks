# Sandicks Telegram Bot

Bot de Telegram para generacion de memes con Python.

## Requisitos

- Python 3.11+
- Token de bot de Telegram
- Variables de entorno en `.env`

## Instalacion local

1. Crear y activar entorno virtual.
2. Instalar dependencias:
   `pip install -r requirements.txt`
3. Configurar `.env`.
4. Ejecutar el bot:
   `python src/main.py`

## Deploy en Render

- Usa `requirements.txt` para instalar dependencias.
- Configura las variables del `.env` como Environment Variables en Render.
